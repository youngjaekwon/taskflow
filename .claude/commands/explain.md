지정한 파일, 함수 또는 모듈을 읽고 동작 원리를 한국어로 설명하라.

## 입력

$ARGUMENTS

## 절차

1. $ARGUMENTS가 비어있으면 "설명할 대상을 지정해주세요. 예: `/explain apps/web/src/routes/+page.svelte`"라고 안내하고 종료한다
2. 지정된 파일 또는 경로를 읽는다
3. 다음 구조로 설명한다:
   - **목적**: 이 코드가 존재하는 이유
   - **핵심 로직**: 주요 흐름과 알고리즘을 단계별로 설명
   - **의존성**: 임포트하거나 의존하는 모듈/라이브러리
   - **데이터 흐름**: 입력 → 처리 → 출력 흐름
4. 코드 블록을 인용하며 설명한다

## 행동 규칙

- 설명은 반드시 **한국어**로 작성한다
- 코드를 수정하지 않는다. 읽기 전용으로만 동작한다
- 지나치게 길지 않게, 핵심에 집중한다
